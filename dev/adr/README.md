# ADRs

Architecture and design decisions that guide ongoing development of Quantroot.

Use `kebab-case` filenames with a trailing ISO date, e.g.,
`validator-wrapper-boundary-2026-04-01.md`.

Create a new ADR when a decision changes the long-term structure, boundaries,
or operating model of the stack. Move retired or superseded ADRs to
`../archive/`.
