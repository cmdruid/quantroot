# Reports

Generated reports about the current codebase.

Use `kebab-case` filenames with a trailing ISO date, e.g.,
`rpc-latency-investigation-2026-04-01.md`.

Reports inform follow-on planning and architecture work. Convert active
findings into documents under `plans/` or `adr/`, and archive reports when
no longer current.
