Get test coins for quantum-insured wallet development. Requires a qpub key from the Quantroot soft fork.

How to get a qpub:
bitcoin-cli createwallet "quantum"
bitcoin-cli createsphincskey
bitcoin-cli exportqpub → copy the Q1... key
